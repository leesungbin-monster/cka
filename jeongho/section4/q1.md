## Service 생성하기

### Solution

```
# 1. deployment 확인
k get deploy -n sp-culator

# 2. pod 확인
k get pod -n sp-culator

# 3. service 확인
k get svc -n sp-culator

# 4. deployment 수정
k edit deploy -n sp-culator

# 5. container port 설정 (공식문서 참고)
spec:
      containers:
      - image: nginx:1.14.2
        imagePullPolicy: IfNotPresent
        name: nginx
        ports:
        - containerPort: 80 <- 추가
          protocol: TCP     <- 자동생성

# 6. service 생성 (공식문서 참고 NodePort 검색)
k apply -f - <<EOF
apiVersion: v1
kind: Service
metadata:
  name: front-end-svc   <- 이름 수정
  namespace: sp-culator <- 중요
spec:
  type: NodePort        <- 잊지말것
  selector:
    app: front-end      <- deployment에서 확인
  ports:
    - port: 80          <- 문제에서 요구하는 port 맞춰주기
    - targetPort: 80    <- container port와 맞춰주기
EOF

# 7. 
```
