## Ingress에서 Gateway로 전환

### Solution

```
# 1. gatewayclass 이름 확인
k get gatewayclass

# 2. ingress 조회
k get ingress

# 3. ingress 내용 조회
k get ingress web -o yaml

# 4. ingress 내용 및 문제 요건에 맞게 gateway,HTTPRoute 생성 (공식문서 참조)
k apply -f - <<EOF
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: web-gateway
spec:
  gatewayClassName: nginx
  listeners:
  - name: https
    protocol: HTTPS
    port: 443
    hostname: "gateway.web.k8s.local"
    tls:
      mode: Terminate 
      certificateRefs:
      - kind: Secret
        group: ""
        name: web-cert
---                              # <-- 이거로 리소스 구분지어서 한번에 여러 개 생성가능
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: web-route
spec:
  parentRefs:
  - name: web-gateway
  hostnames:
  - "gateway.web.k8s.local"
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /
    backendRefs:
    - name: web
      port: 80
EOF

# 적용 결과
gateway.gateway.networking.k8s.io/web-gateway created
httproute.gateway.networking.k8s.io/web-route created

# 5. 적용 확인
k get svc -A

# web-gateway-nginx를 보면 됨
NAMESPACE       NAME                              TYPE        CLUSTER-IP       EXTERNAL-IP     PORT(S)                   AGE
default         web-gateway-nginx                 NodePort    10.106.58.54     <none>          443:31251/TCP             14s
ingress-nginx   ingress-nginx-controller          ClusterIP   10.106.171.9     192.168.56.40   80/TCP,443/TCP            32d

# 6. curl 테스트 후 ingress 삭제
k delete ingress web
```

### Result

```
# gateway 조회
[cka0001@k8s-master ~]$ curl -k https://gateway.web.k8s.local:31251
hello

# ingress로 조회
[cka0001@k8s-master ~]$ curl -k https://ingress.web.k8s.local
hello

# ingress 삭제 후 다시 조회
[cka0001@k8s-master ~]$ k delete ingress web
ingress.networking.k8s.io "web" deleted from default namespace

[cka0001@k8s-master ~]$ curl -k https://ingress.web.k8s.local
<html>
<head><title>404 Not Found</title></head>
<body>
<center><h1>404 Not Found</h1></center>
<hr><center>nginx</center>
</body>
</html>

```
