### Ingress 생성하기

### Solution

```
# 1. deploy 조회
k get deploy -n echo-sound

# 2. service 조회
k get svc -n echo-sound

# 3. ingress 존재 유무 조회 (아직 없음)
k get ingress -n echo-sound

# 4. ingressclass 조회 (class 이름 확인 -> nginx)
k get ingressclass -n echo-sound

[cka0001@k8s-master ~]$ k get ingressclass -n echo-sound
NAME    CONTROLLER             PARAMETERS   AGE
nginx   k8s.io/ingress-nginx   <none>       32d

# 5. ingress 생성 (공식문서 참조)
k apply -f - <<EOF
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: echo
  namespace: echo-sound
spec:
  ingressClassName: nginx
  rules:
  - host: example.org # host 안주면 모든 dns에 대하여 다 허용
    http:
      paths:
      - path: /echo # 그냥 루트(/)로 줘도됨 그럼 도메인 이름으로
                    # 오는 모든 문자에 매칭이 됨
        pathType: Prefix
        backend:
          service:
            name: echoserver-service
            port:
              number: 8080
EOF
```

### Result

```
[cka0001@k8s-master ~]$ curl -o /dev/null -s -w "%{http_code}\n" http://example.org/echo
200 # 200 나오면 성공
```
