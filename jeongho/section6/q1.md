## Helm으로 ArgoCD 배포하기

### Solution

```
# 1. helm 조회
helm repo list # command 모를 땐 -h 옵션으로 검색
Error: no repositories to show

# 2. helm repo에 list 추가 (저장소 등록만 하는 것, 차트는 안가져옴)
helm repo add argo https://argoproj.github.io/argo-helm/
"argo" has been added to your repositories
 
# 3. repo에서 list 조회
helm search repo argo
# 뭐 많이 조회됨

# 4. helm template 생성
 helm template argocd argo/argo-cd -n argocd --version 8.6.4 --set crds.install=false > ~/argo-helm.yaml
 
# 5. helm 설치
helm install argocd argo/argo-cd -n argocd --version 8.6.4 --set crds.install=false --create-namespace

# 6. 배포 확인
k get pods -n argocd
```
