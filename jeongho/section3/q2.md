## PVC 복구하기

### Solution

```
# 1. PV 확인
k get pv

# 2. StorageClass 확인
k get sc # 어차피 pv 조회 했을 떄 보이긴 함

# 3. PVC 존재 확인하기 (현재 존재X)
k get pvc -n mariadb

# 4. PVC 생성 (공식문서 참조)
k apply -f - <<EOF
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mariadb
  namespace: mariadb
spec:
  accessModes:
    - ReadWriteOnce
  volumeMode: Filesystem
  resources:
    requests:
      storage: 250Mi
  storageClassName: local-path
EOF

# 5. PVC 조회
k get pvc -n mariadb

# 6. ~/mariadb-deploy.yaml 수정
vi ~/mariadb-deploy.yaml

volumes:
        - name: mysql-persistent-storage
          persistentVolumeClaim:
            claimName: "mariadb"  # <- claimName 추가
            
# 7. deployment 배포
k apply -f mariadb-deploy.yaml

# 8. 파드 확인
k get pod -n mariadb
```
