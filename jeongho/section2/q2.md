## HPA 생성하기

---

### Solution

```
# 1. pod 조회
k get pods -n autoscale

# 2. hpa 조회
k get hpa -n autoscale # 존재하지 않아 아무것도 뜨지 않음

# 3. hpa 존재하지 않으므로 생성(공식문서 참조)
k autoscale deployment apache-server -n autoscale --cpu=50% --min=1 --max=4

# 4. hpa 조회
k get hpa -n autoscale

# 5. downscale stabilization window 설정
k edit hpa -n autoscale

spec 하위에 behavoir 추가(공식문서 참조)
behavior:
  scaleDown:
    stabilizationWindowSeconds: 30
```

### Result

```
# 재조회 시 default로 생성되는 값과 함께 확인 완료
spec:
  behavior:
    scaleDown:
      policies:
      - periodSeconds: 15
        type: Percent
        value: 100
      selectPolicy: Max
      stabilizationWindowSeconds: 30

```
