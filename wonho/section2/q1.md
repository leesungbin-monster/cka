## Nginx config 수정 및 변경 


### Solution
```
k get all -n nginx-static # 리소스 확인
k get cm -n nginx-static -oyaml # cm 내용 확인
k edit cm -n nginx-static nginx-static
->        ssl_protocols TLSv1.3 TLSv1.2; # TLS 설정 추가
k rollout restart deployment/nginx-static # 파드 재기동
```
